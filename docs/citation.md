If you use this book in your research or teaching, please cite it as:

---

## BibTeX

```bibtex
@book{shaw2025causal,
  author    = {Shaw, Charles},
  title     = {Causal Inference in Marketing: Panel Data and Machine Learning},
  year      = {2025},
  publisher = {Self-published (Community Review Edition)},
  url       = {https://github.com/shawcharles/Causal-Inference-in-Marketing},
  note      = {Available at https://shawcharles.github.io/Causal-Inference-in-Marketing/}
}
```

---

## APA Style

Shaw, C. (2025). *Causal Inference in Marketing: Panel Data and Machine Learning* (Community Review Edition). https://github.com/shawcharles/Causal-Inference-in-Marketing

---

## Chicago Style

Shaw, Charles. 2025. *Causal Inference in Marketing: Panel Data and Machine Learning*. Community Review Edition. https://github.com/shawcharles/Causal-Inference-in-Marketing.
